# 🍽️ Chapeau Restaurant Ordering System

A restaurant management and ordering system built with **ASP.NET Core MVC**.

This application allows restaurant staff to manage tables, take orders, send items to the kitchen and bar, and process payments efficiently.

---

## 🚀 Features

* Employee login system
* Table overview and management
* Create and edit orders
* Menu filtering (Lunch / Dinner / Drinks)
* Kitchen and bar workflow
* Order status tracking
* Stock visibility
* Payment handling
* Split bill functionality
* Customer feedback and tips

---

## ⚙️ Setup

1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/Chapeau.git
```

2. Open the project in **Visual Studio**

3. Restore dependencies

```
dotnet restore
```

4. Run the application

```
dotnet run
```

---

## 🎯 Purpose of the Project

The goal of this project is to simulate a real restaurant workflow where staff members can:

* manage tables
* take and update orders
* send items to the kitchen or bar
* monitor order progress
* process payments and tips
* keep the restaurant workflow organized

This system helps reduce order mistakes and improves communication between service staff, kitchen staff, and bar staff.
